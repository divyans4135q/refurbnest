# RefurbNest 📱✨
### Premium Certified Refurbished Electronics Platform

RefurbNest is a high-end full-stack e-commerce marketplace for certified pre-owned electronics. It features a professional, modern aesthetic with staff-picked deals, rigorous 49-point quality auditing, and a seamless shopping experience.

## 🔥 Features
- 💎 **Premium UI/UX:** Built with React 19, Framer Motion, and Tailwind-inspired custom glassmorphism.
- ⚡ **Lightning Fast:** Optimized with React Lazy Loading, Suspense, and Skeleton screens.
- 🛠️ **49-Point Check:** Interactive diagnostic status for every product.
- 💳 **Secure Payments:** Integrated with Razorpay for safe transactions.
- ☁️ **Cloud Database:** Real-time data sync with MongoDB Atlas.
- 📱 **Responsive:** Fully mobile-friendly design.

---

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Zustand, Lucide React, Framer Motion.
- **Backend:** Node.js, Express, Mongoose.
- **Database:** MongoDB Atlas.
- **Cloud Assets:** Cloudinary (Images), Razorpay (Payments).

## ⚙️ Local Setup
1. **Clone the repo:** `git clone https://github.com/divyans4135q/refurbnest.git`
2. **Backend:** 
   - `cd backend`
   - `npm install`
   - Create `.env` with `MONGO_URI`, `JWT_SECRET`, `PORT`.
   - `npm run dev`
3. **Frontend:**
   - `cd frontend`
   - `npm install`
   - `npm run dev`
